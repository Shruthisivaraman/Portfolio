# Shruthi S M — Modern Developer Portfolio

A premium, interactive, and high-fidelity portfolio website engineered for modern web browsers. Tailored specifically for showing expertise in MERN stack development and Artificial Intelligence/Machine Learning engineering.

## 🔗 Live Site
Deploy link: [https://shruthisivaraman.github.io/Portfolio/](https://shruthisivaraman.github.io/Portfolio/)

---

## 🛠️ Tech Stack & Key Features

### 1. Interactive AI "Neural Experience Network"
- Mapped academic qualifications and internship milestones as pulsing input neurons in an interactive network graph.
- Traces bezier SVG synapses that light up connected skill outputs on click.

### 2. Interactive "Honeycomb Skill Hive"
- A custom sci-fi hexagonal interlocking grid housing the full technical skillset (Java, JavaScript, MERN, Tools, and Work Style).
- Features an interactive micro-telemetry console typing descriptions in real-time on hover.

### 3. Interactive "Glass Medal Showcase"
- Highlights academic and professional credentials (NPTEL, Infosys Springboard, Novitech, DOS Club) inside a high-tech glass showcase cabinet.
- Clicking medals triggers a green laser-scanning animation decrypting validated metadata.

### 4. Apple-Style "Sticky Scroll Reveal" Projects
- Side-by-side split screen reveal showcasing featured projects (IBM Payroll system, Aqua Pure business portal, AI Habit Tracker, and Road Accident Prevention).
- Features dynamic scroll intersection observers tracking active visuals.

### 5. Interactive 3D Business Card
- A virtual executive card that dynamically tilts in 3D space tracking mouse movements.
- Radial glow aura light effects and one-click copy contact indicators.

### 6. Direct Mail Delivery (EmailJS Integration)
- Connects the contact form directly to the developer's inbox (`shruthisivaraman25@gmail.com`) with animated toast status feedbacks.

## 🚀 Local Setup

To run this repository locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/Shruthisivaraman/Portfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Portfolio
   ```
3. Open `index.html` in your web browser.
